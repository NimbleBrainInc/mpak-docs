---
title: mpak bundle show
description: Display detailed information about a bundle
---

Get detailed information about a specific bundle, including versions, platforms, and install instructions.

## Usage

```bash
mpak bundle show <package>
```

## Examples

```bash
mpak bundle show @nimblebraininc/echo
mpak bundle show @nimblebraininc/echo --json
```

## Output

```bash
$ mpak bundle show @nimblebraininc/echo

@nimblebraininc/echo (1.0.0)
A simple echo MCP server for testing

Author:   NimbleBrain
Type:     python
License:  MIT
Homepage: https://github.com/NimbleBrainInc/mcp-echo

Downloads: 1,250

Versions:
  1.0.0  darwin-arm64, darwin-x64, linux-x64, linux-arm64
  0.9.0  darwin-arm64, linux-x64

Install:
  mpak bundle pull @nimblebraininc/echo
  mpak bundle run @nimblebraininc/echo
```

## Options

| Option | Description |
|--------|-------------|
| `--json` | Output as JSON |

## Provenance

If the bundle was published via GitHub Actions OIDC, the output includes provenance information:

```
Provenance:
  Repository: NimbleBrainInc/mcp-echo
  Workflow:   release.yml
  Commit:     abc1234
```

This cryptographically proves which repository the bundle came from.
