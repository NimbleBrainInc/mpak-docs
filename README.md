# mpak Documentation

Documentation for [mpak](https://mpak.dev), the package registry for agent capabilities.

Built with [Astro Starlight](https://starlight.astro.build).

## Development

```bash
npm install
npm run dev
```

Open [localhost:4321](http://localhost:4321) to view the docs.

## Build

```bash
npm run build
```

Output is in `./dist/`.

## Deployment

Deployed to https://docs.mpak.dev
